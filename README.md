🍜 午餐转盘（Lunch Wheel）

一个基于 纯 HTML + CSS + JavaScript 实现的午餐决策转盘，用于解决“今天吃什么”的终极难题。

✨ 功能特点
🎯 1. 权重随机机制
每个餐厅支持不同权重
当前配置：
前 5 个：权重 2（≈16.7%）
负一楼自选：权重 1（≈8.3%）
使用加权随机算法，保证概率真实
🎡 2. 真实转盘逻辑（核心亮点）
指针固定在顶部（0°）
转盘旋转对齐结果
👉 视觉结果 = 实际结果

不是简单随机，而是：

随机权重 → 计算角度 → 转盘对齐 → 输出结果
🎨 3. UI 设计
conic-gradient 实现饼图
SVG 精致指针
渐变配色 + 阴影层次
中心轴 + 立体感设计
响应式适配（移动端可用）
⚙️ 4. 动画系统
自定义缓动曲线：
cubic-bezier(0.15, 0, 0.15, 1)
惯性减速（模拟真实转盘）
停止后“轻微反馈动画”（celebrate）
📊 5. 历史记录
显示最近 5 次选择
用于避免连续重复选择
🧩 技术实现
模块	技术
UI 渲染	HTML + CSS
饼图	conic-gradient
动画	CSS transition
指针	SVG
随机算法	JavaScript 加权随机
布局	Flex + Absolute

🧠 核心逻辑说明
加权随机算法
random = Math.random() * totalWeight

逐个累加权重，命中区间即选中

转盘对齐逻辑
finalAngle = 360 - 命中扇区角度

👉 指针始终指向正确结果

转动机制
至少转 5 圈 + 随机圈数 + 精确对齐
